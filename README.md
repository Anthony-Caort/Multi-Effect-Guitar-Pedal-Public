# Multi-Effect-Guitar-Pedal-Public

This is a public version of the final project I worked on during the second half of spring quarter. Currently its based off of the bare_bones version I worked on where the lcd screen driver was simplified for time.

The main processing for audio is done on an IceSugar Pro FPGA, using the PMOD I2S2 module for an ADC and DAC, and a Raspberry Pi Pico 2W to handle the screen logic.
